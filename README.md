# OKWhen-SamsungLabo

Startowy projekt Android Studio (szablon "Navigation Drawer Activity") utworzony na potrzeby ćwiczeń z Samsung Labo.

Zawiera domyślny szkielet wygenerowany przez Android Studio (`MainActivity`, nawigacja boczna, `FloatingActionButton`) bez dodatkowej logiki biznesowej — punkt wyjścia do dalszych zadań z kursu.

## Wymagania

- Android Studio
- Gradle (wrapper dołączony w repo)
